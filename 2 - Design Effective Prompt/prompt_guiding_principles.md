# Give clear and specific instructions

## Use Delimiters

Triple quotes: """
Triple backticks: ```
Triple dashes: ---
Angle brackets: <>
XML tags: <tag></tag>

## Specify a structured output

HTML, JSON

## Check conditions

Check whether conditions are satisfied
Check assumptions required to do the task

## Few-shot prompting
- Provide a few sucessful examples to complete a task
- Ask the model to perform the task
